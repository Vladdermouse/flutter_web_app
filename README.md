# Flutter Web App

Easy WebApp Widget built in Flutter.  
Works on iOS and Android.

Usage example:


```dart
library myweb.app;

import 'package:flutter/material.dart';
import 'package:flutter_web_app/flutter_web_app.dart';

void main() => runApp(
    WebApp(
        url: 'https://apphosted.url/app', 
        statusBarColor: Colors.red
        )
    );
```
